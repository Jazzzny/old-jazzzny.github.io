Discord Legacy

I'm the sole developer for the Discord Legacy project. It aims to patch the Discord desktop client to run on unsupported versions of macOS, ranging from 2014-2017.

jdk-macos-legacy

I'm the sole developer for the jdk-macos-legacy project. It aims to backport the newest versions of the Java Development Kit to unsupported versions of macOS, ranging from 2009-2017.
